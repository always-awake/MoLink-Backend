# MoLink-Backend
모여라!! 링크(MoLink)

- IDE : IntelliJ
- FrameWork : Spring(SpringBoot)
- Deploy : AWS(EC2, RDS, S3, IAM)
- DB : mysql, JPA
- Social Login(Facebook, KakaoTalk) : Firebase
- Proxy(WebServer) : Nginx(Docker에 이미지로 만든 후 ec2에 배포)
